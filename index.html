Context

OMNI Crypto is a new crypto currency company that want to use an authentication software for the onboarding of its new customers. During this onboarding they want to set up a flow where the user receives a simple tutorial to take a picture of the ID, the front and back will be captured and then a selfie will be taken to look for a match with the ID picture. The idea of Omni Crypto is that their new customers could do the onboarding in their website or/and in their android/iOS application.

Solution

Implementation of ID VERIFICATION Incode performs numerous safe checks to verify customers documents are real and up-to-date, and our customizable workflows are able to access multiple government databases for additional verification.

How ID verification Works

1) Capture ID and Biometrics - Capture ID and biometrics in real-time

2) Verify ID - Verify ID is valid (including liveness detection without extra facial movements) and person is physically present

3) Verify Biometrics - against ID

4) Verify Id Validity - Verify validity of ID ( date, format, etc )

4) KYC check - Verify customer-provided information against system of record/kyc check

5) Grant or Deny Access

To provide support to the Android and IOS devices Incode develop and support the Android and IOS SDK

Technical Capacities

Incode’s Omni Platform offers a variety of integration approaches ranging from zero integration using Incode’s self-service portal, to deep integrations through SDKs and APIs. Web, React Web, IOS, Android, React Native, Flutter, Xamarin

The integration of the of the solution can be implemented with three different ways the no integration, client integration, and server to server, where the no integation the user needs to send all the data via SMS.

The client integration where Incode have two ways with a simple web implementation bsaed on REACT JS and the second option with SDKs

And the last option where all is managed with the API ( cloud managed by Incode)

Technical Benefits

The Incode Omni onboarding solution enables fast, reliable, and trusted validation of users personal information, biometrics, and documentation while keeping the highest degree of privacy.

The benefits on the model is the user can decide if the implementation is managed by Incode or the solutions can be hosted by the Client, on this option the Client needs to have or manage a good infrastructure to support the bandwith of the operations, and with this solution Incode Provide licenses to use the solutions
